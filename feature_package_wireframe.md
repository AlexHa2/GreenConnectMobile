
# Wireframe UI/UX Package Dashboard

## Package Dashboard Tổng hợp

```
+------------------------------------------+
| [AppBar] Package Dashboard               |
+------------------------------------------+
| [Section] Thông tin tổng quan            |
|   - Số điểm hiện tại                     |
|   - Gói đang sử dụng / trạng thái        |
|                                          |
| [Button] Mua gói                         |
| [Button] Xem danh sách gói               |
| [Button] Lịch sử dùng điểm               |
| [Button] Lịch sử mua gói                 |
|                                          |
| [Section] Gói nổi bật / Đề xuất          |
|   - Card gói: Tên, giá, mô tả ngắn       |
|   - [Button] Mua ngay                    |
+------------------------------------------+
```

### Điều hướng
- Tất cả chức năng chính đều có button trên dashboard.
- Khi bấm vào button sẽ chuyển sang màn hình tương ứng (mua gói, xem lịch sử, xem chi tiết gói...).
- Có thể quay lại dashboard dễ dàng từ các màn hình con.

---
Wireframe này giúp tập trung trải nghiệm người dùng, thao tác nhanh, dễ quản lý các chức năng liên quan đến package.
